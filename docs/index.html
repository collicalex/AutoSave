Description

AutoSave is a simple portable application to automatically save (backup) your computer files to your external storage drive or to your online cloud storage. For now it support the follwing cloud services:

  • pCloud

It does only incremental backup: all new files will be saved; but files which do not more exist on your local computer will not be deleted from the backup (usb drive).

File Encryption

Autosave encryption had been removed since the last release.

To securely encrypted your backup file, you can:

  • On local storage (usb disck for example)
    • either use a hardware encrypted drive which are now not really expensive (see the Western Digital My Passport (for Mac))
    • either create a virtual encrypted disk by using VeraCrypt
  • On cloud storage
    • Just use the encryption mecanism of your cloud provider.
